# Austin Donovan, Data Analyst Portfolio

Personal portfolio site for Austin Donovan, a data analyst in Orlando, FL. A single-page site with an interactive Q&A panel, project case-study modals with recorded Power BI dashboard demos, and a downloadable resume.

## Featured work

- **E-Commerce Performance and Customer Segmentation** (Olist, 100K+ orders): MySQL, SQL window functions, RFM segmentation in Python, Power BI + DAX.
- **Spotify Charts: Dynamics and Streaming Concentration** (26M rows): chunked Pandas, SQL window functions, four-page Power BI dashboard.
- **Marketing A/B Test: Ad Campaign Conversion** (588,101 users): two-proportion z-test, chi-square, bootstrap confidence intervals.
- **Product Segmentation with K-Means Clustering** (4,479 SKUs): feature engineering, k-means, benchmarked against ABC/Pareto.

## Tech

HTML, CSS, and JavaScript on a small component runtime. Fonts (Geist) are self-hosted. React, ReactDOM, and Babel load from unpkg.com at runtime.

## Run locally

Open `index.html` in a browser. An internet connection is required, since the runtime dependencies load from a CDN.
